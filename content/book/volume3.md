## 第一篇：数学先哲

### The Mathematical Prophets

> *他们定义了我们思考的"语言"。没有他们的抽象，AI 不可能存在。*

|  节  | 人物                                          | 贡献领域                                                   |
| :---: | --------------------------------------------- | ---------------------------------------------------------- |
| §476 | **Leonhard Euler** (1707–1783)         | 分析学、图论、数论——数学的莫扎特                         |
| §477 | **Carl Friedrich Gauss** (1777–1855)   | 数论、最小二乘法、高斯分布——"数学王子"                   |
| §478 | **Bernhard Riemann** (1826–1866)       | 黎曼几何、黎曼积分、Riemann 猜想——高维几何的先知         |
| §479 | **Henri Poincaré** (1854–1912)        | 拓扑学、混沌理论、庞加莱猜想——最后的全才数学家           |
| §480 | **David Hilbert** (1862–1943)          | 希尔伯特空间、形式主义、23 个问题——20 世纪数学的总设计师 |
| §481 | **Emmy Noether** (1882–1935)           | 抽象代数、Noether 定理——对称与守恒的深刻联系             |
| §482 | **Andrey Kolmogorov** (1903–1987)      | 概率论公理化、复杂性理论、湍流——概率论的奠基人           |
| §483 | **John von Neumann** (1903–1957)       | 博弈论、计算机架构、量子力学——跨领域的超级天才           |
| §484 | **Alexander Grothendieck** (1928–2014) | 代数几何革命、范畴论——重写了数学的基础语言               |
| §485 | **Cédric Villani** (1973–)            | 最优传输、Boltzmann 方程——Fields 奖得主与 OT 的布道者    |

## 第二篇：AI 拓荒者

### Pioneers of Artificial Intelligence

> *从图灵的梦想到 LLM 的现实，他们铺就了通向机器智能的道路。*

|  节  | 人物                                  | 贡献领域                                        |
| :---: | ------------------------------------- | ----------------------------------------------- |
| §486 | **Alan Turing** (1912–1954)    | 可计算性、图灵测试——AI 与计算科学之父         |
| §487 | **Claude Shannon** (1916–2001) | 信息论——用比特定义了"信息"的本质              |
| §488 | **Judea Pearl** (1936–)        | 因果推断、贝叶斯网络、do-calculus——图灵奖得主 |
| §489 | **Vladimir Vapnik** (1936–)    | VC 维、SVM、统计学习理论——学习理论的奠基者    |
| §490 | **Geoffrey Hinton** (1947–)    | 反向传播、Boltzmann 机、深度学习——"AI 教父"   |
| §491 | **Yann LeCun** (1960–)         | CNN、自监督学习、JEPA——视觉智能的建筑师       |
| §492 | **Yoshua Bengio** (1964–)      | 深度网络训练、注意力、GAN 理论——图灵奖三巨头  |
| §493 | **Andrej Karpathy** (1986–)    | 视觉识别、自动驾驶 AI、AI 教育——代码即教程    |
| §494 | **Ian Goodfellow** (1985–)     | 生成对抗网络 (GAN)——一夜之间改变生成模型      |
| §495 | **Ilya Sutskever** (1985–)     | AlexNet、Seq2Seq、GPT 系列——OpenAI 首席科学家 |
